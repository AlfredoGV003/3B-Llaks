Manual de Ejecución Rápida
Sigue estos tres pasos para instalar las dependencias y ejecutar la aplicación.

1. Requisitos Previos (Instalación de Software)
Asegúrate de tener instalado el siguiente software:

Node.js y npm: Necesario para que el servidor Express pueda funcionar.

Puedes descargarlo e instalarlo desde el sitio web oficial de Node.js(https://nodejs.org/es).

2. Instalación de Dependencias
Abre la terminal de tu Visual Studio Code.

Navega hasta la carpeta donde descargaste y descomprimiste el proyecto (por ejemplo, cd C:\Usuarios\TuNombre\Descargas\3B-Llaks).

Ejecuta el siguiente comando para instalar todas las librerías necesarias (como Express y SQLite3):

npm install
3. Ejecución de la Aplicación
Asegúrate de que la terminal aún esté dentro de la carpeta del proyecto.

Inicia el servidor Node.js:

node server.js
